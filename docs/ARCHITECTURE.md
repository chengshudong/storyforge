## Clean Architecture Layers

```
api в”Җв”ҖвҶ’ service в”Җв”ҖвҶ’ repository в”Җв”ҖвҶ’ domain
  в”Ӯ                                  в”Ӯ
  в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ infra в—„в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

1. **api** вҖ” HTTP handlers, request/response DTOs
2. **service** вҖ” Business logic, orchestration
3. **repository** вҖ” Data access, transactions
4. **domain** вҖ” Entity models, interfaces
5. **infra** вҖ” External systems (DB, Redis, MinIO, Celery)

No circular imports. Each layer depends only inward.

## Agent System (Phase 2+)

```
Agent в”Җв”ҖвҶ’ Interface в”Җв”ҖвҶ’ Adapter в”Җв”ҖвҶ’ Provider
```

- **Agent**: LangGraph workflow node
- **Interface**: Abstract contract
- **Adapter**: Provider-specific implementation
- **Provider**: External OSS (ComfyUI, CosyVoice, etc.)

All providers isolated, replaceable, containerized, versioned.

## Scene Generation Pipeline

```
Episode вҶ’ Split вҶ’ NГ—Storyboard вҶ’ Validate вҶ’ Save (JSONB)
              вҶ“         вҶ“            вҶ“
           SceneBeat  Camera      Continuity
           Characters  Emotion     IssuesвҶ’WARN
           Duration   Dialogue    (non-blocking)
                      Props
                      AssetRefs
```

- **Split**: 1 LLM call вҖ” episode summary вҶ’ numbered scene beats with characters and duration
- **Storyboard**: N concurrent LLM calls (semaphore=3) вҖ” each beat вҶ’ full cinematography
- **Validate**: 1 LLM call вҖ” checks character continuity, location motivation, time flow, emotional arc, prop consistency
- **Save**: SceneRepository.create() Г— N вҖ” stores storyboard as JSONB column

All LLM calls routed through ModelRouter with timeout (60s), retry (3Г—), fallback chain, and content-addressed Redis caching.
