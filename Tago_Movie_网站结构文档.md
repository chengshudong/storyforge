## 8. 通用组件说明

### 8.1 全局导航栏组件
出现在所有页面顶部，包含以下固定元素：
- Logo 链接 (e0)
- 主菜单链接 (e1-e6)
- 客服按钮 (e7)
- 用户菜单 (e8)
- 升级套餐 (e9)
- 积分显示 (e10)

### 8.2 按钮状态说明
- **disabled**: 按钮禁用状态，不可点击
- **collapsed**: 下拉框收起状态
- **checked**: 单选框选中状态
- **selected**: 标签页选中状态
- **readonly**: 只读状态（如滑块）
- **current**: 当前页面导航项

### 8.3 元素类型说明
- **link**: 超链接，可跳转页面
- **button**: 按钮，执行操作
- **textbox**: 文本输入框
- **radio**: 单选按钮
- **tab**: 标签页
- **slider**: 滑块
- **heading**: 标题（h1-h6）
- **text**: 文本内容
- **img**: 图片
- **region**: 区域容器

## 9. 元素引用规则

在开发过程中，元素引用（ref）遵循以下规则：
- `e` + 数字：如 `e0`, `e1`, `e2`...
- 同一页面内唯一
- 不同页面可以重复使用相同的 ref
- 引用用于在自动化测试或脚本中定位元素

## 10. 开发建议

### 10.1 页面定位
使用 URL 路径进行页面定位：
- 首页：`https://tagomovie.bitaihub.com/app`
- 其他页面：在根路径后添加对应路径即可

### 10.2 元素定位策略
1. **推荐**：使用元素引用 (ref) 进行精确定位
2. **备选**：使用元素名称和类型组合定位
3. **备选**：使用 XPath 或 CSS 选择器

### 10.3 交互流程
1. 导航到目标页面
2. 等待页面加载完成
3. 使用 ref 定位目标元素
4. 执行点击、输入等操作
5. 验证操作结果

### 10.4 注意事项
- 某些按钮有初始禁用状态（如"开始创作"）
- 下拉框需要先点击展开才能选择选项
- 搜索功能需要先输入再点击搜索按钮
- 部分页面需要登录才能访问完整功能
