## 十一、元素交互流程

### 11.1 首页创作流程

```
1. 输入故事想法
   - 点击 e12（故事创作输入框）
   - 输入故事内容
   - 状态变为 active

2. 选择创作模式
   - 点击 e19（整剧模式）- 默认选中
   - 或点击 e20（逐集模式）
   - 或点击 e21（画布模式）
   - 选中状态变为 checked

3. 配置创作设置
   - 点击 e14（选择画面比例）- 展开下拉框
   - 点击 e15（选择创作类型）- 展开下拉框
   - 点击 e16（选择单集时长）- 展开下拉框
   - 点击 e17（选择集数）- 展开下拉框
   - 点击 e18（选择画面风格）- 展开下拉框

4. 开始创作
   - 当所有必填项完成后
   - e22（开始创作）按钮变为可用
   - 点击开始创作按钮
```

### 11.2 项目管理流程

```
1. 搜索项目
   - 点击 e31（搜索框）
   - 输入项目名称或描述
   - 点击 e32（搜索按钮）
   - 显示搜索结果

2. 添加新项目
   - 点击 e33（添加项目）
   - 跳转到项目创建页面

3. 查看项目详情
   - 点击 e36（项目卡片链接）
   - 跳转到项目详情页面
```

### 11.3 资产管理流程

```
1. 查看资产
   - 点击 e33（所有资产）
   - 或点击 e35（角色）
   - 或点击 e36（场景）
   - 或点击 e37（道具）
   - 或点击 e38（音色）
   - 显示对应类别的资产

2. 新建资产
   - 点击 e39（+ 新建角色）
   - 或点击 e40（+ 新建场景）
   - 或点击 e41（+ 新建道具）
   - 或点击 e42（+ 新建音色）
   - 打开资产创建对话框

3. 配置API
   - 点击 e31（API配置页面）
   - 跳转到API配置页面
```

### 11.4 推广中心流程

```
1. 切换标签页
   - 点击 e31（邀请有礼）
   - 或点击 e32（平台合伙人）
   - 或点击 e33（专属贴牌定制）
   - 显示对应标签页内容

2. 邀请好友
   - 点击 e35（复制邀请链接）
   - 复制链接到剪贴板
   - 分享给好友

3. 补填邀请人
   - 点击 e34（/ 补填邀请人）
   - 展开输入框
   - 输入邀请人信息

4. 提取积分
   - 当有可提取积分时
   - 点击 e39（提取积分按钮）
   - 执行积分提取
```

## 十四、开发建议和最佳实践

### 14.1 元素定位策略

**推荐方法**:
1. **使用元素引用 (ref)**: 最准确和稳定
   ```javascript
   // 示例
   browser.click('e12') // 点击故事创作输入框
   ```

2. **使用元素名称和类型组合**: 适用于快速定位
   ```javascript
   // 示例
   browser.click('button[name="联系客服"]')
   ```

3. **使用CSS选择器**: 适用于复杂场景
   ```javascript
   // 示例
   browser.click('button.start-btn')
   ```

### 14.2 等待策略

```javascript
// 等待页面加载
await page.waitForLoadState('networkidle')

// 等待元素可见
await page.waitForSelector('[ref="e12"]')

// 等待元素可点击
await page.waitForElementState('[ref="e22"]', 'enabled')

// 等待URL变化
await page.waitForURL('**/projects')
```

### 14.3 交互最佳实践

```javascript
// 1. 点击前等待元素可交互
await page.click('e12')

// 2. 输入前清空文本框
await page.fill('e31', '')

// 3. 选择单选按钮
await page.check('e19') // 整剧模式

// 4. 处理下拉框
await page.click('e14') // 展开下拉框
await page.selectOption('e14', '16:9') // 选择选项

// 5. 等待按钮可用后点击
await page.waitForSelector('e22:not([disabled])')
await page.click('e22')
```
